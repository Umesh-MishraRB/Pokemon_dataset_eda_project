# Pokémon Data Analysis

A data analysis project exploring the world of Pokémon using Python, data visualization, and statistical analysis techniques.

---

## 📌 Project Overview

This project analyzes Pokémon datasets to discover patterns, trends, and insights such as:

- Strongest Pokémon based on stats
- Type distributions
- Legendary vs non-legendary comparisons
- Correlation between attributes
- Generation-wise analysis
- Attack, Defense, Speed, and HP trends
- Data visualization using graphs and charts

The project demonstrates practical applications of:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Statistical Insights
- Feature Analysis

---

## 📂 Dataset

The dataset contains information about Pokémon including:

- Name
- Type 1
- Type 2
- HP
- Attack
- Defense
- Speed
- Special Attack
- Special Defense
- Generation
- Legendary Status

Example dataset source:

- Kaggle Pokémon Dataset

## 📊 Analysis Performed

### 1. Pokémon Type Distribution
Understanding which Pokémon types are most common.

### 2. Stat Comparisons
Comparing:
- Attack vs Defense
- Speed vs HP
- Legendary vs Normal Pokémon

### 3. Correlation Analysis
Finding relationships between numerical attributes.

### 4. Top Powerful Pokémon
Identifying Pokémon with highest total stats.

### 5. Generation Analysis
Analyzing Pokémon trends across generations.

## 🚀 How to Run the Project

### Clone the repository

```bash
git clone https://github.com/your-username/pokemon-data-analysis.git
```

### Navigate to project folder

```bash
cd pokemon-data-analysis
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run colab Notebook

```bash
colab notebook
```
